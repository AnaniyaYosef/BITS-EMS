# CustomTkinter Setup Instructions

## Installation

To run this application with CustomTkinter, you need to install the required dependencies:

```bash
pip install -r requirements_customtkinter.txt
```

## Dependencies

- `customtkinter==5.2.2` - Modern GUI framework for Python
- `mysql-connector-python==8.2.0` - MySQL database connector

## Running the Application

After installing the dependencies, run the application:

```bash
python hr_app.py
```

## Features

The application provides a modern, professional interface for managing employee data with:

- Clean, responsive UI design
- Professional color scheme
- Modern button and input styling
- Improved user experience with CustomTkinter components
